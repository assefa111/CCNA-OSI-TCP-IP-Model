# CCNA-OSI-TCP-IP-Model
OSI Model and TCP/IP Model  using Cisco Packet Tracer

## Project Overview
This project demonstrates practical understanding of the OSI Model and TCP/IP Model

The project focuses on:
- OSI 7 Layers
- TCP/IP Layers
- Encapsulation and Decapsulation
- Protocols and Port Numbers
- HTTP/HTTPS Traffic Flow
- DNS Resolution Process
- Data Transfer Through the Internet

---

# Real Scenario

A user opens:

https://www.youtube.com

The request travels through:
- Application Layer
- Transport Layer
- Network Layer
- Data Link Layer
- Physical Layer

before reaching the YouTube server through the internet.

---

# OSI Layers

| Layer | Function |
|---|---|
| 7 | Application |
| 6 | Presentation |
| 5 | Session |
| 4 | Transport |
| 3 | Network |
| 2 | Data Link |
| 1 | Physical |

---

# TCP/IP Model

| TCP/IP Layer | OSI Equivalent |
|---|---|
| Application | Layers 5–7 |
| Transport | Layer 4 |
| Internet | Layer 3 |
| Network Access | Layers 1–2 |
